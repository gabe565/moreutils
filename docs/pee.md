## pee

Tee standard input to pipes

```
pee command... [flags]
```

### Options

```
  -h, --help                     help for pee
      --no-ignore-sigpipe        Do not ignore write errors
      --no-ignore-write-errors   Do not ignore SIGPIPE errors
  -v, --version                  version for pee
```
