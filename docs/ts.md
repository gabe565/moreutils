## ts

Timestamp standard input

```
ts [format] [flags]
```

### Options

```
  -h, --help          help for ts
  -i, --increment     Timestamps will be the time elapsed since the last log
  -s, --since-start   Timestamps will be the time elapsed since start of the program
  -v, --version       version for ts
```
